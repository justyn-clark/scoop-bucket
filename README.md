# Scoop Bucket for SMALL CLI

This is the official Scoop bucket for the SMALL CLI on Windows.

## Installation

```powershell
scoop bucket add small https://github.com/justyn-clark/scoop-bucket
scoop install small
```

## Upgrade

```powershell
scoop update small
```

## Uninstall

```powershell
scoop uninstall small
scoop bucket rm small
```

## Available Manifests

| Manifest | Description |
|----------|-------------|
| small    | SMALL CLI - Command line interface for the SMALL protocol |
